# 手动回归测试清单

## 1. 文档目的

这份文档记录当前主线已完成功能里，仍建议统一手动回归的关键测试 case。

适用范围：

* `P0-a`：最小用户归属底座
* `P1-A / P1-B / P1-C`：统一学习主路径、闭环与长期反馈
* `P0-b`：本地备份与恢复 MVP

## 2. 测试前提

建议准备两类账号：

* 账号 A：已有词库、计划、知识点、复习记录
* 账号 B：空白或接近空白的数据账号

建议准备两类数据状态：

* 至少一个激活中的学习计划
* 至少一份可上传恢复的本地备份 zip

## 3. P0-a 账号与归属

### Case A-1 注册新账号

步骤：

1. 打开 `/login`
2. 切到“注册”
3. 输入新用户名、显示名、密码
4. 提交后观察跳转

预期：

* 注册成功
* 自动进入已登录状态
* 跳转到 `/dashboard`

### Case A-2 登录已有账号

步骤：

1. 从未登录状态打开 `/login`
2. 输入已有账号用户名和密码
3. 提交

预期：

* 登录成功
* `/api/me` 可正常返回当前用户
* 页面进入 `/dashboard`

### Case A-3 路由鉴权守卫

步骤：

1. 未登录时直接访问 `/dashboard`
2. 未登录时直接访问 `/backups`

预期：

* 被重定向到 `/login`
* 登录后能回到原目标页或主工作台

### Case A-4 账户资料修改

步骤：

1. 登录后打开 `/account`
2. 修改显示名
3. 切换默认学习顺序
4. 保存

预期：

* 成功提示出现
* Header 和账户页显示最新资料
* 刷新后数据仍正确

### Case A-5 修改密码

步骤：

1. 在 `/account` 输入当前密码与新密码
2. 提交修改
3. 退出登录
4. 使用新密码重新登录

预期：

* 密码修改成功
* 旧密码不能再登录
* 新密码可以登录

### Case A-6 登出

步骤：

1. 在任意已登录页面点击“退出登录”
2. 再次访问受保护页面

预期：

* 会话失效
* 被要求重新登录

## 4. P1 主学习路径与闭环

### Case P1-1 工作台主入口

步骤：

1. 登录后访问 `/`
2. 观察重定向与工作台内容

预期：

* 自动跳到 `/dashboard`
* 能看到单词线与知识点线摘要

### Case P1-2 单词会话自动选计划

步骤：

1. 账号下存在至少一个激活计划
2. 打开 `/cards`

预期：

* 页面自动进入当前可用计划的当日会话
* 如果当前计划无效，会回退到第一个激活计划

### Case P1-3 知识点复习会话

步骤：

1. 打开 `/notes/review`
2. 对至少一条知识点完成评分

预期：

* 页面能连续推进到下一条
* 最近历史和剩余队列正常更新

### Case P1-4 闭环导出入口

步骤：

1. 完成一段单词或知识点复习
2. 按页面收尾引导进入 `/export-jobs`
3. 创建一条导出任务

预期：

* 可成功创建导出任务
* 历史导出记录能展示
* 成功状态任务可下载

### Case P1-5 薄弱项页面

步骤：

1. 打开 `/weak-items`
2. 如果存在薄弱项，尝试移出一条

预期：

* 页面能正常展示单词/知识点薄弱项
* 移出后列表与摘要刷新

## 5. P0-b 本地备份与恢复

### Case B-1 创建整体备份并下载

步骤：

1. 登录后打开 `/backups`
2. 点击“创建备份并下载”

预期：

* 浏览器开始下载 zip
* 页面出现成功提示
* 免费版不展示服务端托管历史列表

### Case B-2 恢复准备成功路径

步骤：

1. 在 `/backups` 上传有效备份 zip
2. 点击“上传并准备恢复”

预期：

* 成功返回恢复准备结果
* 页面展示安全快照文件名
* “确认覆盖恢复”在下载安全快照前保持禁用

### Case B-3 安全快照前置约束

步骤：

1. 完成恢复准备
2. 不下载安全快照，直接观察确认按钮状态

预期：

* 前端按钮保持禁用
* 不允许跳过安全快照直接确认恢复

### Case B-4 完整恢复链路

步骤：

1. 上传有效备份 zip
2. 准备恢复
3. 下载安全快照
4. 确认覆盖恢复
5. 刷新相关业务页面

预期：

* 恢复成功
* 词库、计划、知识点等用户资产按备份内容恢复
* 登录态不丢失

### Case B-5 非法备份包

步骤：

1. 上传非 zip 或损坏 zip
2. 尝试准备恢复

预期：

* 页面显示明确错误信息
* 不进入可确认恢复状态

## 6. 当前未纳入本轮手测范围

* 管理员后台
* 角色/权限模型
* JWT / API token
* 云端托管备份
