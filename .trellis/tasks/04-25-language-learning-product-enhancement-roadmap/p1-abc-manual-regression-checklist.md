# P1-A / P1-B / P1-C 手点回归单

## 1. 回归目标

验证下面三项任务在前端交互层已经串起来：

* `P1-A` 统一学习主路径
* `P1-B` 闭环呈现与完成反馈
* `P1-C` 长期反馈与复盘

---

## 2. 前置准备

### 2.1 账号前置

* 已能登录
* 已能进入 `/dashboard`

### 2.2 最小数据准备

如果当前没有单词和知识点数据，先准备这批最小数据：

1. 创建 1 个词库
2. 往词库里加 3 到 5 个词条
3. 创建 1 个学习计划并激活
4. 创建 3 到 5 个知识点

### 2.3 可选准备：触发闭环层级

如果要验证 `P1-B` 的回捞 / 薄弱轮表达，再额外准备：

1. 进入 `/cards`
2. 对同一张卡多次点 `AGAIN`
3. 进入 `/notes/review`
4. 对同一条知识点多次点“不会”

这样才能看到：

* 回捞 / 恢复队列
* 薄弱轮
* `/weak-items` 数据

## 3. P1-A 回归

### 3.1 无数据场景

1. 打开 `/dashboard`
2. 确认出现“回归前置准备”
3. 确认能看到：
   * 单词线还没起
   * 知识点线还没起
4. 点击按钮能跳到：
   * `/study-plans`
   * `/word-sets`
   * `/notes`

### 3.2 有数据场景

1. 打开 `/account`
2. 切换默认学习顺序：
   * `单词优先`
   * `知识点优先`
3. 回到 `/dashboard`
4. 确认 “Today's Path” 的推荐动作随偏好变化
5. 如果偏好线有待办，确认主按钮跳到对应会话
6. 如果偏好线做完、另一条还有待办，确认出现后续衔接动作
7. 如果两条主线都清空：
   * 没有薄弱项时，显示今日完成
   * 有薄弱项时，显示去 `/weak-items` 的入口

### 3.3 导航分层

1. 查看左侧导航
2. 确认已经拆成：
   * 学习
   * 管理
3. 确认 `dashboard / cards / notes/review / weak-items` 在学习组
4. 确认 `study-plans / word-sets / notes / templates / export-jobs` 在管理组

---

## 4. P1-B 回归

### 4.1 单词会话层级表达

1. 进入 `/cards`
2. 确认右侧提示区显示当前层级
3. 主队列时应看到：
   * “主队列”
   * 为什么这是主复习流
4. 如果触发 `AGAIN` 回捞：
   * 应出现“回捞队列”
   * 应解释为什么会回到当天流程
5. 如果触发薄弱轮：
   * 应出现“薄弱轮”
   * 应解释这是主队列结束后的补强层

### 4.2 知识点会话层级表达

1. 进入 `/notes/review`
2. 确认右侧提示区显示当前层级
3. 主队列时应看到：
   * “主队列”
4. 如果触发 `AGAIN`：
   * 应出现“恢复队列”
5. 如果触发多次 `AGAIN`：
   * 应出现“薄弱轮”

### 4.3 会话完成反馈

单词和知识点两条线都要分别检查：

1. 把当前队列做到完成
2. 确认完成提示里至少能出现这些动作中的合理子集：
   * 回工作台
   * 去另一条学习线
   * 去薄弱项
   * 导出复盘材料

### 4.4 弱项页

1. 打开 `/weak-items`
2. 无数据时：
   * 应显示“当前没有薄弱项”
   * 应能看懂为什么现在没有数据
3. 有数据时：
   * 应看到闭环说明
   * 应能跳回 `/dashboard`
   * 应能跳到 `/export-jobs?source=closure`

### 4.5 导出收尾

1. 从会话完成提示进入导出页
2. 确认 `/export-jobs` 顶部提示你“刚完成一段学习”
3. 确认表单会尽量预填：
   * `planId`
   * `targetDate`
4. 无计划时，应显示“还没有可导出的学习计划”
5. 无历史记录时，应显示“还没有历史导出记录”

## 5. P1-C 回归

### 5.1 无数据场景

1. 打开 `/dashboard`
2. 查看“近 7 天复盘”
3. 确认无数据时不会报错
4. 应出现：
   * 暂无高峰日
   * 近 7 天还没有可复盘的数据

### 5.2 有数据场景

1. 至少完成几次单词和知识点复习
2. 回到 `/dashboard`
3. 检查“近 7 天复盘”：
   * 单词复习量
   * 知识点复习量
   * 有学习记录的天数
   * 当前薄弱项
4. 检查“高峰日”：
   * 单词线高峰
   * 知识点线高峰
5. 检查“风险信号”：
   * 今日是否有积压
   * 是否有激活计划今天未启动
6. 检查“近 7 天明细”表：
   * 日期
   * 单词
   * 知识点
   * 合计

---

## 6. 重点风险

手点时优先盯这几类问题：

* 偏好切换后，主路径推荐没变化
* 会话完成后，跳转按钮把人带回错误页面
* 无数据时页面空白，或只显示 0 但不给解释
* 导出页从收尾入口进入后，没有预填上下文
* 弱项页和工作台对“今日完成”的定义不一致
